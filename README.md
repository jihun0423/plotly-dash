# plotly-dash

인턴을 하며 다른 직원분들의 작업을 보던 도중 시각화가 매우 깔끔하게 정리되어 있었고, 상호작용 가능한 그래프가 그려져 있어서 여쭤봤더니 plotly와 dash를 이용해서 dashboard를 생성한거라고 한다.


평소 시각화 툴이라면 matplotlib이나 가끔 태블로를 사용한 정도라서, 흥미를 가져 공부해 보기로 하였다.
